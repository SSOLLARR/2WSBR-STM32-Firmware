# 2WSBR-STM32-Firmware
# 1. Title
Two-Wheeled Self-Balancing Robot (STM32 Firmware)

## 2. Introduction
This repository contains the embedded C firmware for a Two-Wheeled Self-Balancing Robot (2WSBR). The software is responsible for reading raw inertial data from an MPU6050 sensor, estimating the robot's tilt angle using a complementary filter, and driving two TMC2209 stepper motors via a cascaded PID control loop to maintain an upright position.

## 5. How to Run the Software
1. Clone this repository and open the project in STM32CubeIDE.
2. Build the project and flash it to the STM32 via ST-Link.
3. Ensure the robot is held perfectly vertical (0 degrees) before powering on the 12V battery.
4. The system will perform a 3-second gyro calibration routine upon startup. Do not move the robot during this phase.
5. Release the robot.

## 6. Technical Details
* **Angle Estimation:** A complementary filter is used to fuse the accelerometer and gyroscope data. The loop runs at XX Hz with an alpha tuning parameter of 0.98.
* **Control System:** A discrete-time cascaded PID controller is implemented. 
  * The outer loop calculates the target velocity based on the angle error.
  * The inner loop adjusts the stepper motor step-rate to match the target velocity.
* **Motor Control:** The TMC2209 drivers receive step/dir pulses generated by hardware timers on the STM32.
